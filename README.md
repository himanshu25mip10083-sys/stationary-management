# stationary-management
Project Title
Stationary Management System (Tkinter Python Application)
Overview of the Project
A GUI-based application created using Python’s Tkinter library. 
It allows users to input quantities of stationery items and automatically calculates the total bill.
Features
• Simple Tkinter GUI  
• Fixed price list display  
• Auto calculation of total bill  
• Reset functionality  
• Error-handled inputs
Technologies/Tools Used
Python 3.x, Tkinter
Steps to Install & Run the Project
1. Install Python 3  
2. Save the script as stationary_management.py  
3. Run: python stationary_management.py  
4. GUI window will open
Instructions for Testing
• Enter any quantity  
• Click TOTAL  
• Click RESET to clear  
• Try invalid inputs—handled safely
